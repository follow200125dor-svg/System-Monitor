# 🖥 System Monitor

Real-time system monitor for Windows. Shows CPU, RAM, Disk usage and Network speed.

---

## 📊 Features

| Monitor | Shows |
|---------|-------|
| **CPU** | Usage %, Cores count, History graph |
| **RAM** | Usage %, Used/Total GB, History graph |
| **Disk C:** | Usage %, Free/Total GB |
| **Network** | Download/Upload KB/s |

---

## 🛠 How to Install

### Automatic
1. Double-click `install.cmd`
2. Wait — it installs Python 3.14.2 + psutil
3. Program starts automatically

### Manual
1. Install Python 3.14.2 from [python.org](https://python.org)
2. Open CMD and type:
pip install psutil
python system_monitor.py

text

---

## 🚀 How to Use

1. Launch `system_monitor.py`
2. Watch CPU, RAM, Disk, Network in real-time
3. Green bars = normal, Red bars = high load
4. History graphs show last 100 seconds
5. Close window to exit
